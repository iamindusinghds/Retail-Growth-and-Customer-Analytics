# Retail Customer Intelligence Dashboard

## Project Overview

This project presents an end-to-end retail analytics solution built using Power BI. The dashboard provides insights into sales performance, customer behavior, customer lifetime value (CLV), segmentation, and retention trends.

The objective was to transform retail transaction data into actionable business insights through KPI reporting, customer analytics, and cohort-based retention analysis.

---

## Key Features

### Executive Dashboard

* Total Revenue Analysis
* Total Orders Tracking
* Total Customers Monitoring
* Average Customer Lifetime Value (CLV)
* Monthly Revenue Trends
* Regional Sales Performance
* Product Category Performance

### Customer Analytics Dashboard

* Customer Segmentation Analysis
* Customer Cluster Distribution
* Customer Lifetime Value (CLV) Analysis
* High-Value and At-Risk Customer Identification

### Cohort Retention Analysis

* Monthly Cohort Tracking
* Customer Retention Matrix
* Retention Pattern Analysis
* Customer Lifecycle Insights

## Business Insights

* Identified top-performing regions contributing the highest revenue.
* Analyzed customer segments based on purchasing behavior.
* Evaluated Customer Lifetime Value (CLV) distribution across customer groups.
* Measured customer retention trends using cohort analysis.
* Built interactive dashboards for business decision-making.
